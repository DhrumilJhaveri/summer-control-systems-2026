Summer Control Systems & Robotics 2026

This repository documents my Summer 2026 learning journey as a BEng Mechatronics and Robotics student at the University of Sheffield.

Goals
1) Develop proficiency in Python and MATLAB for engineering applications
2) Strengthen understanding of feedback control systems
3) Design and simulate dynamic systems
4) Build and control a self-balancing robot
6) Learn the fundamentals of Model Predictive Control (MPC)
7) Create a portfolio of engineering projects and technical reports

Planned Timeline
1) Weeks 1–2: Python, MATLAB, and control fundamentals
2) Weeks 3–4: Self-balancing robot hardware and PID control
3) Weeks 5–7: Mathematical modelling and simulation
4) Weeks 8–10: Introduction to MPC
5) Weeks 11–12: Embedded implementation, documentation, and final report
